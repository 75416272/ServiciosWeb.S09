# ServiciosWeb.S09

Proyecto desarrollado en ASP.NET Core para practicar el uso de servicios web mediante rutas API y una página HTML sencilla.

## Descripción

El sistema permite realizar consultas básicas usando condicionales desde una interfaz web.  
El usuario ingresa datos en formularios simples y el sistema muestra una respuesta según la opción seleccionada.

## Funcionalidades

- Clasificar edad.
- Validar acceso de usuario.
- Calcular descuento según tipo de cliente.
- Mostrar el día de la semana según un número ingresado.

## Tecnologías utilizadas

- ASP.NET Core
- C#
- HTML
- JavaScript
- Visual Studio

## Estructura principal del proyecto

- `Program.cs`: configuración principal del proyecto.
- `Controllers/CondicionalesController.cs`: controlador con las rutas API.
- `wwwroot/index.html`: página principal del sistema.
- `README.md`: descripción del proyecto.
- `.gitignore`: archivos y carpetas que no deben subirse al repositorio.

## Cómo ejecutar el proyecto

1. Abrir el proyecto en Visual Studio.
